**⚠️ MrDoc API 问题记录 (2026-03-25 最终解决)**:
- **问题**: 使用 `/api/create_doc/` + `editor_mode=1` 上传 HTML，MrDoc v9.3 有时显示 HTML 源码而非渲染页面
- **根本原因 (2026-03-25 23:42 确认)**: **CSS 复杂度问题**
  - ❌ 复杂 CSS（gradient、flex、box-shadow 等）会导致 MrDoc 无法渲染，显示源码
  - ✅ 基础 CSS（简单颜色、边框、padding）可以正常显示
- **解决方案**: 
  - 使用简化版 HTML 模板（参考 `/tmp/fund_report_2026-03-20.html` 格式）
  - 避免使用：linear-gradient、flexbox、box-shadow、复杂 CSS 选择器
  - 只使用：基础颜色、边框、padding、margin、简单表格样式
- **验证结果**: doc-94（简化版）✅ 正常显示，doc-95（完整版）✅ 正常显示
- **技能文档已更新**: `skills/mrdoc-api-guide.md`
- **经验教训 (2026-03-25)**:
  1. 上传前必须测试 HTML 兼容性
  2. 使用已验证的模板格式，不要随意添加新 CSS
  3. 出现问题立即删除错误文档，避免混淆
