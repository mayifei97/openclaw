## New API 问题排查
- 问题：New API 使用日志对普通用户显示 upstream_model_name（实际模型 astron-code-latest），暴露了模型重定向信息
- 原因：logs 表 other 字段存储了 upstream_model_name，前端日志详情页对所有用户都展示"Model Mapping"区域
- 前端代码逻辑：`n?.is_model_mapped && n?.upstream_model_name` 时显示，无权限判断
- 解决方案：需修改 New API 源码，在日志 API 返回时对非管理员过滤掉 upstream_model_name 和 is_model_mapped
- Docker 镜像 `calciumion/new-api:latest` 是编译好的二进制，无法直接修改
- 要自行修改需：git clone 源码 → 修改 → go build → 替换
