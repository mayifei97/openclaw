## 🔴 MrDoc API 问题 - 已解决 ✅

**问题描述**: MrDoc API 多个路径均返回 404

**根本原因**: MrDoc v9.3 的 API 参数名与之前版本不同
- 旧参数：`project_id`, `content`
- 新参数：`pid`, `doc`

**解决方案**: 
1. 重启 MrDoc Docker 容器：`docker restart mrdoc`
2. 使用正确的 API 端点：`/api/create_doc/?token=xxx`
3. 使用正确的参数名：`pid`, `title`, `doc`, `editor_mode`
