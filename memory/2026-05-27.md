### 完整安装顺序（9 个包）
```
1. pip install setuptools-82.0.1-py3-none-any.whl
2. pip install pycparser-2.22-py3-none-any.whl
3. pip install cffi-1.17.1-cp312-cp312-win_amd64.whl
4. pip install bcrypt-4.3.0-cp38-abi3-win_amd64.whl
5. pip install cryptography-44.0.3-cp37-abi3-win_amd64.whl
6. pip install PyNaCl-1.5.0-cp36-abi3-win_amd64.whl
7. pip install paramiko-5.0.0-py3-none-any.whl
8. pip install pyyaml-6.0.3-cp312-cp312-win_amd64.whl
9. pip install win11toast-0.35-py3-none-any.whl
```

### 依赖关系
```
pycparser → cffi → bcrypt/cryptography/PyNaCl → paramiko
setuptools（独立）
PyYAML（独立）
win11toast（独立）
```
