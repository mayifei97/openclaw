## 三、逐基金深度分析与操作纪律

### 🔵 人工智能(012733) — 核心争议持仓

**数据面：**
- 当前净值2.3224，近1年+116%，1年分位98%
- 60日波动率2.39%，属于高波动品种
- 近20日涨+9.4%，距60日高点仅-0.9%

**我的独立判断：**

你说AI还在高速发展阶段，这个我部分认同。**但我要指出一个关键风险：AI行业增速和市场估值是两回事。**

当前事实：
- 中证人工智能指数1年涨116%，处于98%历史分位，**估值已极度昂贵**
- AI大模型竞争白热化（字节、鸿海、SK等巨头持续加码），但**商业化变现仍在早期**
- 芯片/算力板块今日+1.8%领涨，短期资金仍在流入
- 6G概念启动（工信部6G创新发展专项行动），可能接力AI炒作

**结论：AI长期看好，但短期处于泡沫区。** 你已减仓1/3是正确的，剩余仓位不应该加仓，应该继续锁利。98%分位意味着只有2%的时间比现在更贵，这时候加仓是赌它继续涨，概率上不划算。

**操作纪律：**

| 条件 | 净值区间 | 操作 | 逻辑 |
|------|---------|------|------|
| 极度超买 | ≥ 2.45 | 🔴 再减1/3（剩~570元） | 1年分位98%+，泡沫风险大 |
| 偏高震荡 | 2.25 - 2.45 | 🟡 观望，不操作 | **当前2.32在此区间** |
| 回调至合理 | 2.05 - 2.25 | 🟢 可考虑加仓500元 | 回调10%+，性价比改善 |
| 深度回调 | ≤ 2.05 | 🟢 加仓500-800元 | 1年分位回到60-70%，有价值 |
| 极端下跌 | ≤ 1.90 | 🟠 减至1/3仓位 | 趋势可能彻底反转 |

**⚠️ 与你意见的差异：** 你认为回调时适合加仓，我同意**但条件更严格**——只有跌到2.05以下（1年分位回到70%以下）才值得加仓，2.25以上加仓风险收益比太差。
