## BSdiff GUI 工具开发

### 用户需求
用户需要一个Windows GUI工具，调用其C代码(test.c)中的`aw_bsdiff()`函数生成bsdiff补丁文件。

**输入参数：**
- 老参数文件路径
- 新参数文件路径
- 输出bsdiff文件路径（默认：新文件目录/isp_param_config.bsdiff）

**最终交付：**
- Windows可执行文件(.exe)
- 无需依赖，可在任意Windows电脑运行

### 设计方案

**Python版本：**
- 使用tkinter创建GUI界面
- 调用编译后的test.exe生成补丁
- 使用PyInstaller打包为单文件EXE

### 文件结构

```
bsdiff_python_gui/
├── test.c          # 用户的C代码（核心算法）
├── bsdiff_gui.py   # Python GUI界面
├── build.bat       # 一键打包脚本
└── README.md       # 使用说明
```

### 使用步骤

1. **安装MinGW-w64**（只需一次）
   - 下载：https://www.mingw-w64.org/downloads/
   - 添加到系统PATH

2. **运行build.bat**
   - 双击运行
   - 自动编译test.c → test.exe
   - 自动打包为BSdiffGUI.exe

3. **使用**
   - 复制dist/BSdiffGUI.exe到任意位置
   - 双击运行，无需其他依赖

### 技术要点

**查找test.exe的多种方式：**
1. PyInstaller打包后的临时目录 (`sys._MEIPASS`)
2. EXE所在目录
3. 脚本所在目录
4. 当前工作目录

**打包命令：**
```bash
pyinstaller --onefile --windowed --name BSdiffGUI --add-data "test.exe;." bsdiff_gui.py
```
