### 问题2: 定时任务"今日执行"状态错误
**原因**: 
- `get_cron_history` 函数使用错误参数 `--job-id` (应为 `--id`)
- `parse_cron_runs` 函数未正确解析 JSON 格式的 `entries` 数组
- Python 3.6 不支持 `datetime.fromisoformat`

**修复**:
- 修改参数: `--job-id` → `--id`
- 重写 `parse_cron_runs` 函数，正确解析 `entries` 数组
- 使用手动解析 ISO 时间格式兼容 Python 3.6

**状态**: ✅ 代码修复完成，待验证

### 操作教训
- 避免使用 `killall` 等影响范围大的命令
- 重启服务前确认 PID 正确
- 使用 `kill -15` 优雅退出而非强制杀死
