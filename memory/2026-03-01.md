### 迈曦软件公司查询

用户询问迈曦软件公司的主营业务。

**遇到的技术限制：**
- web_search 工具缺少 Brave API 密钥
- browser 工具无法连接（Chrome 扩展未绑定标签页）
- 多个网站访问被限制或返回空结果

**解决方法：**
- 使用 SearXNG 本地搜索工具多次尝试不同关键词
- 最终通过 `site:mxcae.com` 搜索找到官网
- 使用 web_fetch 获取官网关于页面信息

**查询结果：**
迈曦软件是湖南大学科技成果转化的 CAE 软件公司，2020 年成立，主营自主可控的工程仿真软件（MxSim 系列），团队 200+ 人，研发占比 85%+。

**备注：**
- 本次查询体现了"三次尝试自主解决准则"——尝试了多种搜索方法和关键词组合
- SearXNG 技能在 web_search 不可用时是有效的替代方案
