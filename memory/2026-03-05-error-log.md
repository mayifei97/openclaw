3. **多数据源对比**: 同时获取多个数据源，选择最新的
4. **人工复核机制**: 重要数据发送前做简单校验

**教训**:
- 基金净值更新时间：一般 20:00-22:00 陆续更新，但可能延迟
- 定时任务执行时间要预留充足缓冲
- 数据准确性比及时性更重要
