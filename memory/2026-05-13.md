## 🗺️ 路线地图生成经验

**用户反馈教训**：
- 用户明确指出：坐标折线图（用 matplotlib 纯画线）**不是真正的地图**
- 用户原话："感觉就是一个表格里面画几根线，都不是真正的地图"
- **正确做法**：必须用真实地图瓦片做底图（高德/百度/Google），在上面叠加路线标注

**成功方案**：
- 高德地图瓦片服务器：`https://webrd01.is.autonavi.com/appmaptile?lang=zh_cn&size=1&scale=1&style=8&x={x}&y={y}&z={z}`
- 该服务器**无需 API Key**即可获取中文标注地图瓦片
- 用 Python PIL 拼接瓦片 + matplotlib 叠加标注，生成 PNG 图片
- 中文显示问题：用 `/usr/share/fonts/truetype/wqy/wqy-microhei.ttc` 字体解决

**注意事项**：
- 跨区域总览（如九龙东 + 沙田）距离较远，瓦片地图缩小后看不清道路细节
- 总览图可以改用坐标示意图，但**单个区域的图必须用真实地图底图**
- 路线坐标需要精确（最好用道路 API 获取轨迹），否则直线连接会偏离实际道路
